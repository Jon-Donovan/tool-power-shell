
# Compress-CurrentProject

Создаёт ZIP‑архив текущего проекта.

## Описание

Функция читает `.compress.config.json`, подставляет placeholders в путь архива,
фильтрует исключённые файлы и формирует ZIP‑архив.

## Используемые поля конфигурации

ArchivePath  
Exclude

## Алгоритм работы

1. Определить корневой каталог проекта
2. Загрузить конфигурацию
3. Разрешить placeholders в ArchivePath
4. Проверить корректность расположения архива
5. Удалить существующий архив при наличии
6. Получить список файлов проекта
7. Отфильтровать файлы по правилам Exclude
8. Добавить файлы в ZIP‑архив
