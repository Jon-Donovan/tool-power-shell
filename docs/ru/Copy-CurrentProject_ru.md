
# Copy-CurrentProject

Копирует текущий проект в целевой каталог.

## Описание

Функция читает `.compress.config.json`, проверяет целевой путь
и рекурсивно копирует файлы проекта, учитывая правила исключений.

## Используемые поля конфигурации

TargetDirectory  
CleanTargetDirectory  
Exclude

## Алгоритм работы

1. Определить корневой каталог проекта
2. Загрузить конфигурацию
3. Проверить корректность целевого каталога
4. Очистить целевой каталог (если указано)
5. Получить список файлов проекта
6. Исключить файлы согласно Exclude
7. Скопировать файлы с сохранением структуры каталогов

## Пример

Copy-CurrentProject
