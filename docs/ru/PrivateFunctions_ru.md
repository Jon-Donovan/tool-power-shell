
# Приватные функции

Модуль содержит внутренние вспомогательные функции,
используемые публичными командами.

## Утилиты путей

Get-NormalizedDirectoryPath  
Get-NormalizedFilePath  
Test-PathInsideDirectory

Используются для безопасного сравнения и проверки путей.

## Фильтрация файлов

Test-Excluded

Проверяет, соответствует ли файл или каталог маске исключения.

## Обработка placeholders

Resolve-Placeholders

Подставляет динамические значения в строки.

## Файловые операции

Copy-ProjectItem

Рекурсивное копирование файлов и каталогов.

## Проверки безопасности

Test-SafeDirectory

Предотвращает опасные операции удаления.
